== Доступ в интернет

Для работы программы в корпоративной сети с ограниченным доступом в интернет, чтобы работали интернет-сервисы программы, вам следует создать следующие правила фаервола. Все доступы являются необязательными, программу можно использовать без доступа в интернет. Читайте описание, чтобы решить, какой список адресов актуален для вашего случая и на работу каких сервисов повлияют ограничения.
